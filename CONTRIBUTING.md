# Katkıda Bulunma Rehberi

Teşekkürler! Bu repoya katkıda bulunmak istiyorsanız lütfen aşağıdaki adımları izleyin:

1. Reponun bir fork'unu oluşturun.
2. Yeni bir feature branch çıkarın: `git checkout -b feature/adiniz-dosya`
3. Değişikliklerinizi küçük, mantıklı commit'lerle gönderin.
4. Fork'unuza push edin ve pull request açın.
5. PR açıklamasında yaptığınız değişiklikleri ve gerekçesini kısa yazın.

Kod/Belgeler için stil önerileri:
- Markdown dosyalarında açık başlıklar ve bölümleme kullanın.
- Özellikle hukuki veya resmi içerik eklerken kaynak/bağlantı gösterin.

Herhangi bir sorunda issue açmaktan çekinmeyin.
